2.1. This app gets recent news and displays them along with the language of the article and allows the user to filter the articles based on the language.

2.2.
Requirements:
Site is deployed through github.io
Site is responsive and utilizes media queries at 768px to change presentation
Utilizes newsdata.io API for acquiring news, detectlanguage API for language detection, and built-in Intl API to change language code from detectlanguage API to the English word of the language.
Site uses a search to find relevant news.
Site displays articles relevant to the search along with the language of the article.
Results are filterable by the language of the articles.
